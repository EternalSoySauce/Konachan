<div>
    <h4><font color='#006600'>Types</font></h4>
    There are six types of tags with the following sort hierarchy:
    <span><br/><font color='#DD00DD'>· Copyright</font></span>
    <span><br/><font color='#00AA00'>· Character</font></span>
    <span><br/><font color='#CCCC00'>· Artist</font></span>
    <span><br/><font color='#00BBBB'>· Circle</font></span>
    <span><br/><font color='#FF2020'>· Style</font></span>
    <span><br/><font color='#EE8887'>· General (default type)</font></span>

    <div>
        <h6><font color='#DD00DD'>Copyright</font></h6>
        <p>Copyright tags identify the name of the anime, game, visual novel or some sort of copyrighted setting.</p>
     </div>

    <div>
        <h6><font color='#00AA00'>Character</font></h6>
        <p>Character tags identify the character(s) in the post.</p>
    </div>

    <div>
        <h6><font color='#CCCC00'>Artist</font></h6>
        <p>Artist tags identify the artist. This doesn't mean the artist of the original copyrighted artwork (for example, you wouldn't use the <code><font color='#CCCC00'>takeuchi_takashi</font></code> tag on a picture of Tohsaka Rin drawn by <code><font color='#CCCC00'>kazenokaze</font></code>).</p>
    </div>

    <div>
        <h6><font color='#00BBBB'>Circle</font></h6>
        <p>Circle tags are meta-copyrights. Most are artist circles (artist collectives) or studios like <code><font color='#00BBBB'>key</font></code> which are in a way also artist circles. Also included are publications like <code><font color='#00BBBB'>Megami</font></code> and <code><font color='#00BBBB'>NyanType</font></code>. These are used ONLY for official artwork (produced or published by...) unless otherwise stated in the wiki (eg. <font color='#EE8887'>Nintendo</font> is an exception).</p>
        <p>Konachan also uses circle tags for collective copyrights such as the <code><font color='#00BBBB'>monogatari_(series)</font></code> where it's not an actual copyright in of itself, but rather a group of related copyrights.</p>
    </div>

    <div>
        <h6><font color='#FF2020'>Style</font></h6>
        <p>Style tags are used to tag things that do not fall under one of the other types. These include things like artistic styles, image descriptors, holiday themes and vocaloid songs.</p>
        <p>Vocaloid songs are tagged when an image is created for or contains references to a particular song or it's official artwork. Tags for vocaloid songs should end in <code><font color='#FF2020'>_(vocaloid)</font></code>.</p>
    </div>

    <div>
        <h6><font color='#EE8887'>General</font></h6>
        <p>General tags are used to tag an object (eg. book), action (eg. kiss), environment (eg. landscape) or color (eg. green).</p>
    </div>

    <div>
        <h6><font color='#FFFFFF'>Ambiguous Flag</font></h6>
        <p>Any tag can be flagged <i>ambiguous</i>, that is the <i>ambiguous</i> flag is separate from the tag type. When a search is performed using a tag flagged <i>ambiguous</i>, such will be noted above the results and possible corrections will be listed.</p>
	</div>
</div>
