# K Anime Wallpaper (K动漫壁纸)
一款动漫图片搜索软件，集合了多网站图源，可在线预览、标签/中文搜索，多尺寸下载、分享给好友、设为手机壁纸。

## 最新版本【v1.6.2】
[百度云盘下载](https://pan.baidu.com/s/125mkF_F1kpKaSVrcugXpIw)，提取码: 81as


## 网站支持
目前已支持网站源：
* Konachan
* Yande.re
* Lolibooru
* Danbooru
* Sankaku

后续即将支持网站源：
* Gelbooru
* 3dbooru
* Zerochan
* E-shuushuu
* Xbooru
* The anime gallery
* 其他
